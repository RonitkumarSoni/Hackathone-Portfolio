## 🎯 What Is This?

Not your typical developer portfolio. This is an **interactive AI experience** — visitors chat with **Nova**, my personal AI assistant powered by Google Gemini, who answers questions about my skills, projects, and experience in real-time with streaming responses.

> **Built in a weekend for the WebForge Hackathon** 🏆

---

## ✨ Key Features

| Feature | Description |
|---------|------------|
| 🤖 **AI Chat (Nova)** | Gemini-powered assistant with streaming responses, context memory, and tool-based UI rendering |
| 🎨 **Interactive 3D** | Three.js particle constellation background with mouse-reactive animations |
| 📱 **Fully Responsive** | Pixel-perfect on mobile, tablet, and desktop |
| ⚡ **Blazing Fast** | Next.js 15 + React 19 with ISR and edge-optimized API routes |
| 🏗️ **13 Live Projects** | Auto-fetched from GitHub API with live demo links |
| 📜 **18 Certificates** | Real PDF certificates viewable in-browser |
| 🎭 **Animated Memoji** | WebM/MP4 avatar that reacts to chat interactions |
| 🔍 **SEO Optimized** | OpenGraph, Twitter Cards, semantic HTML |

## 🚀 Quick Start

```bash
# Clone the repo
git clone https://github.com/RonitkumarSoni/Hackathone-Portfolio.git
cd Hackathone-Portfolio

# Install dependencies
npm install

# Set up environment variables
cp .env.example .env.local
# Add your keys to .env.local

# Start development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) 🎉

### Environment Variables

```env
GEMINI_API_KEY=your_gemini_api_key
GITHUB_TOKEN=your_github_token
GITHUB_USERNAME=your_github_username
NEXT_PUBLIC_GITHUB_TOKEN=your_github_token
NEXT_PUBLIC_GITHUB_USERNAME=your_github_username
```
